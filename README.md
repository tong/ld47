# Superposition ~ LD47

http://games.disktree.net/superposition/

## Controls
### Gamepad
* DPad: select atom
* L1: select next electron
* R1: select prev electron
* A: fire electron
### Keyboard
#### P1
* WASD: select atom
* Q: select prev electron
* E: select next electron
* F: fire electron
#### P2
* Arrows: select atom
* B: select prev electron
* N: select next electron
* M: fire electron
